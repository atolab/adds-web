About Cyclone DDS

Cyclone DDS (cdds) is an Open Source implementation of the OMG Data Distribution Service.

You can learn more and contribute on GitHub.

There are already a series of related projects, such as:

  • ocaml-cdds the OCaml binding to cdds
  • python-cdds the Python byindis to cdds
  • dstore-cdds a distributed key-value store implemented on cdds

Have questions or suggestions? Feel free to open an issue on GitHub or ask me on Twitter.

Thanks for reading!
